# 🗺️ GIS Tracker  
### AI-Powered Land Safety Validation System (KGIS)

**Validate land parcels against environmental and legal risks in seconds.**

---

## 📌 Overview

**GIS Tracker** is an AI-powered geospatial validation system built for Karnataka. It integrates with **KGIS (Karnataka GIS)** to analyze land parcels and detect risks related to:

- Water bodies and lake boundaries  
- Forest and eco-sensitive zones  
- Government-acquired or restricted lands  

Users simply provide a **survey number** or **geo-coordinates**, and the system generates a **Safety Report** with risk classification, legal flags, and visual map insights — without requiring GIS expertise.

---

## 🚨 Problem

Land verification today is:
- Manual and time-consuming  
- Requires GIS knowledge  
- Prone to human error  

Buyers often unknowingly purchase land in restricted zones, leading to **legal and financial issues**.

---

## 💡 Solution

GIS Tracker automates:
- Data retrieval from KGIS  
- Multi-layer spatial validation  
- Risk classification  
- Report generation  

➡️ Entire process completes in seconds.

---

## ✨ Key Features

- 🔄 Automatic KGIS layer integration  
- 🔍 Multi-layer spatial validation  
- ⚠️ Risk classification (Low / Medium / High)  
- 📄 Automated safety reports  
- 🗺️ Interactive map visualization  
- 💬 Easy-to-understand explanations  

---

## 🏗️ System Workflow

1. User inputs survey number or coordinates  
2. System retrieves relevant GIS layers  
3. Spatial analysis is performed  
4. Risk is calculated based on overlaps  
5. Safety report is generated with visual output  

---

## 🧠 Core Logic

The system uses:
- Spatial intersection and buffer analysis  
- Rule-based risk scoring  
- Layer-wise validation  

Risk Levels:
- **Low** → No conflicts  
- **Medium** → Near restricted zones  
- **High** → Direct overlap with restricted land
